### findRtan
## 게임구성
1. 제한시간 60초 동안 16장의 카드가 주어진다.
2. 각 카드는 짝이 있다. 즉 8개의 페어로 존재한다.
3. 클릭으로 카드를 뒤집어 그림을 확인 할 수 있으며 짝이 맞지 않는 2개의 카드를 뒤집었을 경우 다시 뒤집어진다.
4. 짝이 맞는 카드를 뒤집었을 경우 해당하는 2개의 카드는 사라진다.
5. 제한 시간 동안 16장의 카드를 모두 없애는 것이 게임의 최종 목표
